# Proyecto Juego - Java
# Descripción del Proyecto
Este proyecto es un juego desarrollado en Java que permite a los jugadores interactuar con un entorno virtual. El juego incluye características como gráficos, sonido y una interfaz de usuario intuitiva. El objetivo es recrear un juego de combate estilo Street Fighter o Mortal Kombat, donde los jugadores pueden elegir entre diferentes personajes y luchar  contra otros jugadores.

# Características del Juego
- **Gráficos**: El juego cuenta con gráficos en 2D que representan a los personajes y el entorno de combate.
- **Personajes**: Los jugadores pueden elegir entre varios personajes.
- **Controles**: El juego utiliza controles simples para moverse, atacar y defenderse.
- **Multijugador**: El juego permite a dos jugadores competir entre sí en el mismo dispositivo.
- **Sonido**: El juego incluye efectos de sonido para mejorar la experiencia de juego.
- **Interfaz de Usuario**: La interfaz es fácil de usar y permite a los jugadores navegar por el menú y seleccionar opciones de juego.
- **Puntuación**: El juego lleva un registro de la puntuación de cada jugador durante el combate.
- **Animaciones**: Los personajes tienen animaciones para ataques y movimientos.

# Requisitos del Sistema
- Java Development Kit (JDK) 21 o superior 
- Un entorno de desarrollo integrado (IDE) como Eclipse, IntelliJ IDEA o NetBeans
- Sistema operativo compatible con Java (Windows, macOS, Linux)
- Recomendado: Tarjeta gráfica compatible con OpenGL para mejorar el rendimiento gráfico
- Recomendado: Al menos 4 GB de RAM para un rendimiento óptimo
- Recomendado: Procesador de al menos 2 GHz para una experiencia de juego fluida

# Instrucciones de Instalación
1. **Descargar el Código Fuente**: Clona o descarga el repositorio del proyecto desde GitHub.
2. **Configurar el Entorno de Desarrollo**: Abre tu IDE favorito y configura un nuevo proyecto Java. Asegúrate de incluir las bibliotecas necesarias para gráficos y sonido.
3. **Compilar el Código**: Compila el código fuente del proyecto para asegurarte de que no haya errores de compilación.
4. **Ejecutar el Juego**: Ejecuta la clase principal del juego para iniciar la aplicación. Sigue las instrucciones en pantalla para seleccionar personajes y comenzar a jugar.
5. **Disfrutar del Juego**: ¡Diviértete jugando y compitiendo contra tus amigos!
